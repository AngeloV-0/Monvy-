MONVY 

Projeto dos alunos do 1º período do curso de Administração da Universidade Estadual da Região Tocantina do Maranhão (UEMASUL), campus Imperatriz. A proposta consiste na criação de uma plataforma simples de educação e controle financeiro, voltada especialmente para pessoas que nunca tiveram acesso a orientações sobre como lidar com o dinheiro
